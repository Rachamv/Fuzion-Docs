Project Configuration (fuzion.toml)

The fuzion.toml file contains project settings:

fuzion.toml 
[package]
name = "my-app"
version = "0.1.0"
author = "Your Name"

[dependencies]
# Add packages here when needed

Welcome to Fuzion!

Fuzion is a modern programming language designed to be simple, safe, and powerful. Whether you're learning to code for the first time or building production applications, Fuzion provides the tools you need without unnecessary complexity.

What Makes Fuzion Different?

  • Read Like English - Code that makes sense at first glance
  • Safe by Default - Prevents bugs before they happen
  • Fast Performance - Compiled to native code
  • Built for Modern Apps - Web, blockchain, and system programming
  • Easy to Learn - Start building in minutes
üí° Perfect for Beginners

Never programmed before? No problem! Fuzion was designed with beginners in mind. You'll write working code faster than you think.

Installation

Choose your operating system and follow the simple installation steps:

Installing on macOS

Option 1: Quick Install (Recommended)

Open Terminal and run:

bash 
curl -fsSL https://get.fuzion-lang.org | sh

Option 2: Using Homebrew

bash 
brew install fuzion
‚ÑπÔ∏è System Requirements

Works on both Intel and Apple Silicon Macs. Requires macOS 10.15 or later.

Installing on Linux

Quick Install (All Distributions)

bash 
curl -fsSL https://get.fuzion-lang.org | sh

Ubuntu/Debian

bash 
sudo apt update
sudo apt install fuzion

Fedora

bash 
sudo dnf install fuzion

Arch Linux

bash 
sudo pacman -S fuzion

Installing on Windows

Option 1: Using winget (Recommended)

Open PowerShell or Command Prompt and run:

powershell 
winget install fuzion

Option 2: Download Installer

1. Download the installer from fuzion-lang.org/download
2. Run the fuzion-installer.exe
3. Follow the installation wizard

‚ÑπÔ∏è System Requirements

Requires Windows 10 or Windows 11. Works on both x64 and ARM64.

Verify Installation

After installation, verify Fuzion is working:

bash 
fuzion --version

You should see something like:

output 
Fuzion 1.0.0
üí° Having Issues?

If the fuzion command isn't found, you may need to restart your terminal or add Fuzion to your PATH. See our troubleshooting guide.

Your First Program

Let's write the classic "Hello, World!" program. This will confirm everything is working.

Create a new file

Create a file called hello.fz (you can use any text editor):

hello.fz 
print "Hello, World!"

That's the entire program! Just one line.

Run your program

Open your terminal in the same directory and run:

bash 
fuzion hello.fz

You should see:

output 
Hello, World!

Try something interactive

Let's make it more personal. Update your file:

hello.fz 
# Ask for the user's name
name = input "What's your name? "

# Greet them personally
print "Hello, {name}! Welcome to Fuzion!"

Run it again:

bash 
fuzion hello.fz

Type your name when prompted!

üéâ Congratulations!

You just wrote and ran your first Fuzion program! Notice how readable the code is - it reads almost like plain English.

Try the Interactive Mode

Fuzion also has an interactive mode (REPL) for quick experiments:

bash 
fuzion play

Then type code and see results immediately:

fuzion-repl 
> name = "Alice"
> print "Hi, {name}!"
Hi, Alice!
> age = 25
> print "You are {age} years old"
You are 25 years old

Type exit or press Ctrl+D to quit.

IDE Setup (Optional)

While you can use any text editor, these editors have special support for Fuzion with features like syntax highlighting, auto-completion, and error checking.

Visual Studio Code (Recommended)

Install VS Code

Download from code.visualstudio.com

Install Fuzion Extension

Open VS Code and press Ctrl+P (or Cmd+P on Mac), then type:

vscode 
ext install fuzion-lang.fuzion

Start Coding!

Create a new .fz file and enjoy features like:

  • Syntax highlighting
  • Auto-completion
  • Error detection
  • Code formatting
  • Integrated terminal

Other Editors

Fuzion also supports:

  • IntelliJ IDEA - Install the Fuzion plugin from the marketplace
  • Vim/Neovim - Install fuzion.vim plugin
  • Sublime Text - Install Fuzion package via Package Control
  • Emacs - Install fuzion-mode

For larger applications, use Fuzion's project system to organize your code and dependencies.
