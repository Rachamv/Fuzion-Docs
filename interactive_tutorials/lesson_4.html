Loops: Repeating Actions

Learn how to make your programs repeat tasks automatically without writing the same code over and over!

Why Do We Need Loops?

Imagine you want to print "Hello" 10 times. Would you write:

The tedious way (don't do this!) 
print "Hello"
print "Hello"
print "Hello"
print "Hello"
print "Hello"
print "Hello"
print "Hello"
print "Hello"
print "Hello"
print "Hello"
# This is exhausting! What if you need 100 times?

That's terrible! Loops let you repeat code automatically:

The smart way (using a loop) 
for i in 1 to 10:
    print "Hello"
Output:
üéØ Key Concept

Loops let you run the same code multiple times without repeating yourself. They're essential for processing lists, counting, and automating repetitive tasks!

The "for" Loop

The for loop is perfect when you know how many times you want to repeat something.

Basic Range Loop

Count from 1 to 5 
for i in 1 to 5:
    print "Count: {i}"
Output:

How it works:

  1. i starts at 1
  2. Runs the code inside the loop
  3. i increases to 2
  4. Runs again... continues until i reaches 5
  5. After 5, the loop stops
üí° Loop Variable

i is called the loop variable. You can name it anything (count, num, x), but i is traditional for "index" or "iteration".

The "while" Loop

Sometimes you don't know exactly how many times to loop. Use while to keep looping as long as a condition is true!

Basic while loop 
change count = 1

while count <= 5:
    print "Count is {count}"
    count = count + 1

print "Done!"
Output:
‚ö†Ô∏è Infinite Loop Warning!

Be careful! If the condition never becomes false, your loop will run forever! 

change x = 1
while x > 0:
    print x
    x = x + 1  # x keeps growing, always > 0. Infinite loop!

Always make sure your loop variable changes to eventually make the condition false!

When to Use For vs While

Situation Use For Loop Use While Loop
Known iterations ‚úì Yes - for i in 1 to 10 Possible, but for is clearer
Unknown iterations Not ideal ‚úì Yes - while condition
Process a list ‚úì Yes - for item in list Possible, but for is easier
Until condition met Not ideal ‚úì Yes - while not done
üí° Quick Rule

Use for: When you know how many times or have a list to process
Use while: When you're waiting for something to happen or a condition to be met

Controlling Loops: Break and Continue

Sometimes you need to exit a loop early or skip certain iterations.

Break: Exit the Loop

Stop when you find it 
for i in 1 to 10:
    print "Checking {i}"

    if i == 5:
        print "Found it! Stopping."
        break

print "Loop finished"
Output:

Continue: Skip to Next Iteration

Skip even numbers 
for i in 1 to 10:
    if i % 2 == 0:
        continue  # Skip even numbers

    print i  # Only prints odd numbers
Output:
üéØ Remember

break: Exit the entire loop immediately
continue: Skip the rest of this iteration and move to the next one

Nested Loops

Multiplication table (all combinations)

A prime number is only divisible by 1 and itself.

Steps:

  2. For each number, check if it's divisible by any number from 2 to (number - 1)
