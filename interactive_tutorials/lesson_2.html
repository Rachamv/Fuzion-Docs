Order of Operations

Fuzion follows standard math rules (PEMDAS/BODMAS):

  1. Parentheses / Brackets
  2. Exponents / Orders (powers)
  3. Multiplication & Division (left to right)
  4. Addition & Subtraction (left to right)
Order matters! 
# Without parentheses
result1 = 5 + 3 * 2
print "5 + 3 * 2 = {result1}"   # 11 (multiplication first)

# With parentheses
result2 = (5 + 3) * 2
print "(5 + 3) * 2 = {result2}" # 16 (addition first)

# Complex calculation
result3 = 10 + 5 * 2 ** 2 - 3
print "10 + 5 * 2 ** 2 - 3 = {result3}" # 27
Output:
üí° Pro Tip

When in doubt, use parentheses () to make your calculations clear!

Integers vs Decimals

Fuzion handles both whole numbers (integers) and decimal numbers (floats) automatically:

Number types

age = 25
year = 2025
print "Age: {age}, Year: {year}"

# Decimals (floating-point numbers)
price = 19.99
temperature = 36.5

# Mixed operations

Updating Variables

Remember from Lesson 1: use change to make variables mutable. Then you can update them using shortcuts:

Shortcut operations 
# Create a changeable variable
change score = 0
print "Starting: {score}"

# Long way
score = score + 10
print "After adding 10: {score}"

# Add more
score = score + 5
print "After adding 5: {score}"

# Subtract
score = score - 3
print "After subtracting 3: {score}"

# Multiply
score = score * 2
print "After doubling: {score}"
Output:
