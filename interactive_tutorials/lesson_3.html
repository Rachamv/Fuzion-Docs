Control Flow: Making Decisions

Learn how to make your programs smart by teaching them to make decisions based on conditions!

What is Control Flow?

So far, our programs run from top to bottom, line by line. But real programs need to make decisions and take different actions based on conditions.

Think about everyday decisions:

  • If it's raining, bring an umbrella
  • If you're hungry, eat food
  • If the light is red, stop
  • If the light is green, go

Programming uses the same logic!

üéØ Key Concept

Control flow lets your program choose what to do based on conditions. It's like giving your code the ability to think!

The "if" Statement

The most basic decision-making tool is the if statement.

Basic Syntax

Simple if statement 
age = 20

if age >= 18:
    print "You can vote!"

print "Program continues..."
Output:

How it works:

  1. if age >= 18: - Check the condition
  2. If TRUE: run the indented code
  3. If FALSE: skip the indented code
  4. Continue with the rest of the program
üí° Indentation Matters!

In Fuzion, indentation (spaces at the beginning) shows what code belongs to the if statement. Use 4 spaces or 1 tab.

The "if-else" Statement

What if we want to do something when the condition is FALSE? Use else!

If-else example 
age = 16

if age >= 18:
    print "You're an adult!"
else:
    print "You're a minor"

print "Have a nice day!"
Output:

How it works:

  • If condition is TRUE: run the first block
  • If condition is FALSE: run the else block
  • One or the other ALWAYS runs (never both, never neither)
üéØ Remember

With if-else, you're guaranteeing that one of the two blocks will execute. It's like a fork in the road - you must take one path or the other!

The "else if" Chain

Need to check multiple conditions? Use else if!

Grade calculator 
score = 75

if score >= 90:
    print "Grade: A"
else if score >= 80:
    print "Grade: B"
else if score >= 70:
    print "Grade: C"
else if score >= 60:
    print "Grade: D"
else:
    print "Grade: F"

print "End of grading"
Output:

How it works:

  1. Check first condition (>= 90)
  2. If FALSE, check next condition (>= 80)
  3. If FALSE, check next condition (>= 70)
  4. Continue until one is TRUE or reach else
  5. Only ONE block runs, then exit
‚ö†Ô∏è Order Matters!

Always put the most specific conditions first! If you check >= 60 before >= 90, a score of 95 would match >= 60 and never reach the >= 90 check.

Combining Conditions

Sometimes you need to check multiple conditions at once. Use logical operators!

Operator Meaning Example When TRUE

not Opposite/negation not is_raining Condition is false

    print "You can enter"

    print "Entry denied"

Nested If Statements

You can put if statements inside other if statements! This is called nesting.

Nested conditions 
age = 25
has_license = true
has_car = true

if age >= 18:
    print "You're old enough"

    if has_license:
        print "You have a license"

        if has_car:
            print "You can drive your car!"
        else:
            print "You need a car"
    else:
        print "You need a license"
else:
    print "You're too young"
Output:
üí° Tip: Keep It Simple

While nesting is powerful, too many levels can make code hard to read. Try to use and/or instead when possible!

One-Line If (Ternary Operator)

For simple conditions, you can write if-else in one line:

Compact if-else 
age = 20

# Long way:
if age >= 18:
    status = "adult"
else:
    status = "minor"

print "Long way: {status}"

# Short way (one line):
status = "adult" if age >= 18 else "minor"

print "Short way: {status}"
Output:

Syntax: value_if_true if condition else value_if_false

üí° When to Use

Use one-line if for simple assignments. If the logic is complex, use regular if-else for better readability!
